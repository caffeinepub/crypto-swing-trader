# Specification

## Summary
**Goal:** Transform the Crypto Swing Trader into a cyberpunk-styled application by removing Portfolio and Journal features and implementing a complete visual redesign with neon aesthetics, glowing effects, and tech-inspired typography.

**Planned changes:**
- Remove all Portfolio feature code, UI components, navigation links, hooks, and backend functions
- Remove all Trading Journal feature code, UI components, navigation links, hooks, and backend functions
- Implement cyberpunk color scheme with deep black backgrounds (#0a0a0f or darker) and neon accent colors (electric blue #00d9ff, vibrant purple #b300ff, neon green #39ff14)
- Add glowing box-shadow effects to interactive elements, cards, buttons, and data displays with pulsing animations for critical alerts
- Update typography to use tech-inspired fonts (geometric sans-serif for headings, monospace for numerical data)
- Add smooth CSS transitions and animations with cyberpunk-style effects (scanlines, glitches, data-streams)
- Redesign CryptoTable with neon price displays, glowing signal badges, and electric accent colors
- Redesign AITradeAnalysisCard with glowing recommendation cards, neon entry points, and pulsing high-confidence indicators
- Redesign Market Heatmap with neon-colored blocks, electric gradients, glowing borders, and click flash animations
- Update Alert History with dark cards, neon borders, glowing signal badges, and scanline effects
- Redesign charts with neon candlesticks, electric gridlines, glowing data points, and vibrant technical indicators
- Update Layout navigation with dark background, glowing links, neon borders, and futuristic mobile menu
- Redesign WatchlistPanel with neon borders, glowing crypto icons, and pulsing price displays
- Update MarketSentimentPanel with glowing indicators, neon fear/greed display, and electric top gainers/losers
- Redesign IndicatorsPanel with neon borders, glowing values, and electric color coding for overbought/oversold zones
- Update toast notifications with dark backgrounds, neon border glows, and type-specific electric accents
- Enhance mobile responsiveness for cyberpunk design while maintaining 60fps performance

**User-visible outcome:** Users experience a futuristic cyberpunk-styled trading application with streamlined features (Portfolio and Journal removed), featuring dark backgrounds, neon glowing effects throughout all components, tech-inspired fonts, smooth animations, and an immersive visual aesthetic while retaining all core crypto tracking, AI analysis, and alert functionality.
